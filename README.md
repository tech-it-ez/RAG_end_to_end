# RAG_end_to_end
A Retrieval-Augmented Generation (RAG) that enables intelligent querying of PDF documents using semantic search and LLM-powered responses.

This project implements a Retrieval-Augmented Generation (RAG) system designed for interacting with PDF documents. It combines semantic search with large language models to deliver accurate, context-aware answers grounded in your data.

Built using Python, LangChain, OpenAI API, and FAISS, the system processes and indexes PDF files into a vector database, allowing users to perform natural language queries through an intuitive Streamlit interface.

Instead of relying solely on pre-trained knowledge, the application retrieves relevant document chunks and augments them with LLM-generated responses—ensuring more reliable and domain-specific answers.
